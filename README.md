# サバオキャッチ 🐟

凶暴なキジトラ猫「鯖男」を操作して、落ちてくる刺身と魚をキャッチするゲームです。

## 🎮 遊び方

落ちてくるアイテムを鯖男の口でキャッチ！

| アイテム | 効果 |
|----------|------|
| 🍣 刺身 | キャッチで +1点 |
| 🐟 魚 | キャッチで +3点 |
| 💩 うんち | 食べるとミス |

❤️ ミスが3回でゲームオーバー。スコアが上がるほど落下速度が上がります。

## 📱 操作方法

| 環境 | 操作 |
|------|------|
| スマホ | 傾けると鯖男が左右に動く |
| PC（トラックパッド・マウス） | なぞると鯖男が動く |
| キーボード | ← → キー（または A / D キー） |

## 😾 鯖男について

ミスをするたびにブチギレます。「シャーッ！！」と叫びながら顔が赤くなり、画面が揺れます。それが普通です。
